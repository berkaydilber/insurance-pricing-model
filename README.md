# Insurance Pricing Model

## Objective
Estimate insurance charges based on customer risk factors.

## Methods
- Python (Pandas, Scikit-learn)
- Linear Regression

## Key Insight
Smoking is the most significant factor:

- Non-smokers: ~8,400
- Smokers: ~32,000
- ~4x higher cost

## Model Performance
R² ≈ 0.78

## Conclusion
Smoking strongly impacts insurance pricing and should be considered in risk segmentation.
